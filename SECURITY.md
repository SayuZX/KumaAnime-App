# Security Policy

## Supported Versions

We actively support and patch the latest release of Kuma Anime. Please ensure you are running the latest version before reporting vulnerabilities.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:
- Do not disclose the vulnerability publicly (e.g., via GitHub issues) until it has been resolved.
- Send a detailed description of the vulnerability to the project maintainers.
- Include step-by-step reproduction instructions, potential impact, and any potential remediation steps.

We will acknowledge your report, investigate the issue, and release a fix as soon as possible.
